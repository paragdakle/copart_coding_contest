Instructions to execute StingToInteger

Kindly note that in case an alphanumeric string is given as input
the number till the first non-numeric character except '-' at start will be converted.

1. Build the java file.
javac StringToInteger.java

2. Run the program as:
java StringToInteger 1234

Sample Executions:
java StringToInteger -1425
-1425

java StringToInteger 1234
1234

java StringToInteger 1234dsadas
1234
